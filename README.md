# Meeting Schedule App

A modern, professional web application built with React, Vite, and TailwindCSS for scheduling and managing meetings efficiently.

## Features

- **Modern UI/UX**: Clean, responsive, and beautiful interface designed with TailwindCSS.
- **Meeting Management**: Easily schedule, edit, and view upcoming meetings.
- **Interactive Date Picking**: Seamless date and time selection using `react-datepicker`.
- **State Management**: Robust and fast state management using `zustand`.
- **Notifications**: Integrated with `react-toastify` for real-time user feedback.

## Tech Stack

- **Frontend Framework**: React 19 + Vite
- **Styling**: TailwindCSS
- **State Management**: Zustand
- **Date/Time Formatting**: Day.js
- **Components**: React Datepicker

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory
   ```bash
   cd Meeting_Schedule_App/my-app
   ```
3. Install dependencies
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm run dev
   ```
